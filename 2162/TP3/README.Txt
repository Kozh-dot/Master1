ATTENTION l'éxecution de ce programme ne fonctionnera complétemment que sur Linux

OS Utilisé:
    Ubuntu 18.04 LTS

Environnement Java:
    java version "11.0.3" 2019-04-16 LTS
    Java(TM) SE Runtime Environment 18.9 (build 11.0.3+12-LTS)
    Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.3+12-LTS, mixed mode)
    javac 11.0.3

Aucune Station ne possède 2 fois la même ligne.
Le réseau ne possède pas 2 fois la même station.

La méthode de recherche du chemin le plus court ne possède pas de limite de trajet
telle qu'on pourrait en mettre si nécessaire, car si l'on utilise un autre fichier metro.txt
avec des trajets en station plus grand que ceux prévues elle ne trouverait aucun chemin.

Étant donné qu'aucune fonction non récursive n'a été réussit elles ont été retiré.

Pour lancer le programme, ouvrez un terminal et placez vous là où se trouve le fichier 
TP3_LEBERRE_SAMUEL.jar ainsi que le fichier metro.txt et tapez la commande suivante :
    java -jar TP3_LEBERRE_SAMUEL.jar
